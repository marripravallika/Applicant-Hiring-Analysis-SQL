* Applicant Hiring Analysis (SQL)

  Overview :
This project analyzes applicant hiring data using SQL to understand
stage-wise progression, drop-offs, conversion rates, and factors
influencing final offer outcomes.

  Dataset :
- 20 applicants
- Hiring stages: applied, test, interview, offer, rejected
- Data includes applicant background and stage progression dates

   Key Insights :
- Interview stage is the primary bottleneck in the hiring process
- Applicants from IIT show higher offer conversion rates
- Early applicants have a higher chance of receiving offers

   Analysis Performed :
- Stage-wise hiring analysis
- Offer conversion rate calculation
- College and degree-wise performance analysis
- Early vs late applicant success comparison
- What-if analysis on interview success improvement

   Skills & Tools :
- SQL (Constraints,Joins, Aggregations,Group by and Having clauses, CASE, Subqueries)
- Analytical Thinking
- Business KPI Analysis
